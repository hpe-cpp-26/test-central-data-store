# Group: Self-Healing System

## Summary

This project focuses on building a resilient distributed backend system capable of automatically detecting failures and recovering without manual intervention. It includes monitoring, anomaly detection, failure simulation, and automated recovery workflows.

## Responsibilities

- Detect service failures and anomalies in real-time
- Trigger automated recovery actions (restart, fallback, reroute)
- Maintain system health metrics and logs
- Simulate failures for testing resilience
- Enable observability across distributed services
