# ml-training-pipeline

## Group Overview
This document group contains technical documentation for the ML Training Pipeline project, focusing on the design and implementation of an experiment tracking system. It covers the architecture, workflows, and tools used to manage machine learning model development, training, and deployment.

## Documents in this Group
- **ML Training Pipeline — Experiment Tracking Design** *(Source: Confluence)*  
  Defines the architecture for dataset versioning, experiment tracking, model registry, and model promotion workflows for production deployment.

## Key Topics
- Dataset versioning
- Experiment tracking
- Model promotion workflow
- MLflow integration
- GPU cluster training
- Model registry
- Feature drift monitoring
- gRPC model serving
